## 💻 Getting Started <a name="getting-started"></a>


The user needs to run the commands inside my_list.rb on irb environment.

- Use [my_list.rb](./my_list.rb) to create the methods.

### Prerequisites

In order to run this project you need to install Ruby.

### Setup
- Clone the project with git clone git@github.com:luis-pomare/enumerable.git

- Execute the methods inside `my_list.rb` to run the program.
